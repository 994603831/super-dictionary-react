### 2. 项目使用方法

- 安装星云链Google插件钱包

安装方法：[https://github.com/ChengOrangeJu/WebExtensionWallet](https://github.com/ChengOrangeJu/WebExtensionWallet)

- 克隆代码

```
$ git clone https://github.com/liyuechun/super-dictionary-react.git
$ cd super-dictionary-react
$ npm install
$ npm start
```
